# HurtowniaDanych-Projekt

Projekt z przedmiotu Hurtownie Danych realizowany na piątym semestrze studiów na WAT.

Projekt został zrealizowany w grupie 3-osobowej i obejmował trzy etapy:

1. Analiza źródła danych pod kątem zapytań analitycznych do raportów oraz model punktowy.
2. Proces ETL (Extract, Transform, Load).
3. Tworzenie kostki OLAP oraz raportów w Power BI.

W repozytorium zamieszczono finalną wersję sprawozdania z tego przedmiotu z opisem poszczególnych kroków.
